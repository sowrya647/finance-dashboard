# Finance Dashboard

A modern, feature-rich finance dashboard built with React.js to help users track and manage their financial activities.

## Features

### Core Features
- Dashboard Overview: Summary cards showing total balance, income, and expenses
- Data Visualization: Balance trend charts and spending breakdown charts
- Transaction Management: Full CRUD operations with search, filter, and sort
- Role-Based UI: Switch between Viewer and Admin roles
- Financial Insights: AI-powered spending analysis and recommendations
- Responsive Design: Works on all screen sizes
- Data Persistence: Local storage for data persistence

### Additional Features
- Interactive charts (Line & Pie charts)
- Advanced filtering and sorting
- Mobile-responsive design
- Bulk delete transactions
- Modern UI with animations
- Role-based access control

## Tech Stack

- React 18 - Frontend framework
- Context API - State management
- Chart.js - Data visualization
- Tailwind CSS - Styling
- Lucide React - Icons
- date-fns - Date formatting
- Vite - Build tool
